# OilNow+ APK 빌드 가이드

## 현재 상황
- ✅ React Native 프로젝트 완성
- ✅ Expo prebuild 완료 (Android 프로젝트 생성됨)
- ✅ 프로젝트 ZIP 파일 준비됨 (1.2MB)
- ❌ 로컬 빌드 환경 없음 (Android SDK 미설치)

## 빌드 옵션

### 옵션 1: Expo Go 앱 (권장 - 테스트용)
**장점:**
- 설치 불필요
- 즉시 테스트 가능
- QR 코드로 공유

**단계:**
1. 가족이 Google Play에서 "Expo Go" 다운로드
2. 앱 실행
3. QR 코드 스캔
4. 앱 실행

### 옵션 2: 온라인 APK 빌더
**사이트:**
- https://www.apkbuilder.online/
- https://buildozer.online/

**단계:**
1. ZIP 파일 업로드
2. 빌드 시작
3. APK 다운로드

### 옵션 3: EAS Build (공식)
**요구사항:**
- Expo 계정 필요
- 이메일 인증 필요

## 추천
**Expo Go + 온라인 APK 빌더 병행**
- Expo Go: 빠른 테스트
- 온라인 빌더: 설치 가능한 APK
